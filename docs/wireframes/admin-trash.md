# ごみ箱管理画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   ごみ箱管理                  │
├───────────────────────────────┤
│ [全件復元][全件完全削除]       │
│ ─────────────────────────── │
│ 種別 | 名前/内容 | 削除日 | 復元 | 完全削除 │
│ ─────────────────────────── │
│ 写真 | IMG_001.jpg | 2025/09/10 | [復元][削除] │
│ コメント | すてきな写真！ | 2025/09/11 | [復元][削除] │
│ ...                                 │
└───────────────────────────────┘
```

- 写真・コメント等の復元・完全削除
- 一括操作も可能
