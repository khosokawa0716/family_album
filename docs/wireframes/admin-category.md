# カテゴリ管理画面(ワイヤーフレーム例)

```
┌───────────────────────────────┐
│ 管理者用メニュー  - カテゴリ管理 - │
├───────────────────────────────┤
│ [＋新規カテゴリ追加]           │
│ ─────────────────────────── │
│ カテゴリ名 | 編集 | 削除        │
│ ─────────────────────────── │
│ 家族旅行   | [編集][削除]      │
│ 誕生日     | [編集][削除]      │
│ ...                                 │
└───────────────────────────────┘
```

- 名称編集、削除
- 削除時、紐づく写真の `category_id` は NULL になる
- 管理者以外の一般ユーザーがアクセスした場合「アクセス権限がありません」などのエラーメッセージ
