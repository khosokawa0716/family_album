# 写真詳細画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   家族のアルバム              │  ← ヘッダー
├───────────────────────────────┤
│         [写真（大きめ表示）]   │
│                               │
│  [ダウンロード] [削除]         │  ← 操作ボタン（権限による表示制御）
├───────────────────────────────┤
│ コメント一覧                  │
│ ┌───────────────┐           │
│ │ ユーザーA: すてきな写真！   │           │
│ │ ユーザーB: ○○で撮影？      │           │
│ └───────────────┘           │
│ [コメント入力欄][投稿]         │
└───────────────────────────────┘
```

- 写真クリックで拡大表示（モーダル等）
- コメントの編集・削除は自分の投稿のみ可
- 管理者は「削除」「復元」等の操作が可能
