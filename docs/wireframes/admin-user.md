# ユーザー管理画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   ユーザー管理                │
├───────────────────────────────┤
│ [＋新規ユーザー追加]           │
│ ─────────────────────────── │
│ ユーザー名 | 権限 | 状態 | 編集 | 削除 │
│ ─────────────────────────── │
│ userA     | 管理者 | 有効 | [編集][削除] │
│ userB     | 一般   | 有効 | [編集][削除] │
│ ...                                 │
└───────────────────────────────┘
```

- 新規ユーザー追加・編集・削除
- 権限変更や状態（有効/無効）切替
