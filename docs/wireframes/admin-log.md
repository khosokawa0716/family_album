# 操作ログ画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   操作ログ                    │
├───────────────────────────────┤
│ 日時       | ユーザー | 操作内容 | 対象   │
│ ─────────────────────────── │
│ 2025/09/10 | userA   | 削除     | IMG_001.jpg │
│ 2025/09/11 | userB   | 投稿     | IMG_002.jpg │
│ ...                                 │
└───────────────────────────────┘
```

- 操作履歴の一覧表示
- フィルタや検索も将来拡張可能
