# 写真投稿画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   家族のアルバム              │  ← ヘッダー
├───────────────────────────────┤
│ [ファイル選択]                │
│ [カテゴリ選択]                │
│ [投稿]ボタン                  │
└───────────────────────────────┘
```

- JPEG/PNG/GIF/WEBP/HEIC/HEIF対応、20MBまで
  - HEIC/HEIFは自動的にPNG形式に変換して保存
- カテゴリはプルダウン選択
- 投稿後は一覧画面へ遷移
