# 写真投稿画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   家族のアルバム              │  ← ヘッダー
├───────────────────────────────┤
│ [ファイル選択]                │
│ [カテゴリ選択]                │
│ [投稿]ボタン                  │
└───────────────────────────────┘
```

- JPEG/PNG/GIF/WEBP対応、10MBまで
  - ※将来的にHEIC対応（初期実装はスキップ）
- カテゴリはプルダウン選択
- 投稿後は一覧画面へ遷移
