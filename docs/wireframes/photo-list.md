# 写真一覧画面（ワイヤーフレーム例）

```
┌───────────────────────────────┐
│   家族のアルバム              │  ← ヘッダー（タイトル・メニュー）
├─────────────┬───────────────┤
│ [カテゴリ選択] │ [年月選択]      │  ← 絞り込みフィルタ
├─────────────┴───────────────┤
│  ┌─────┬─────┬─────┐         │
│  │写真1 │写真2 │写真3 │ ...    │  ← サムネイル一覧（3列）
│  ├─────┼─────┼─────┤         │
│  │写真4 │写真5 │写真6 │ ...    │
│  └─────┴─────┴─────┘         │
│                               │
│  ...（無限スクロール）         │
│                               │
└───────────────────────────────┘
```

- サムネイルをクリックで「写真詳細」へ遷移
- 画面上部にカテゴリ・年月の絞り込み
- ヘッダーに「投稿」「設定」などのメニューアイコン
- 無限スクロールで写真を追加読み込み

---

※draw.io等での図形化も可能です。ご要望があれば他画面も作成します。
